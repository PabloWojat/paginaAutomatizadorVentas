¿Qué es una landingPage?

Una LandingPage es una página de una cara colgada a la web como una típica página web con la diferencia que una LandingPage es solo para cumplir una función, Puede que sea mostrar las cualidades de tu producto para generar interés en tu posible cliente y convencerlo para asi venderle, puede que muestres las información necesaria de tu servicio para que te contraten, pero siempre generando una sola función, esta página tiene un diseño realizado con una estrategia donde el cliente tiene varios botones que todos redirigen a un lugar, puede que sea a tu WhatsApp, a tu e-commerce, a tú página de gestión de turnos o al punto donde culmina la comunicación tuya con tu cliente.

Que es un Hosting ?

El hosting, o alojamiento web, es un servicio donde podrás alojar todos los contenidos necesarios (textos, imágenes, bases de datos, emails…) para que tu web y correo sea accesible por cualquier usuario en internet en cualquier momento. Del mismo modo que un hotel aloja a sus huéspedes en habitaciones, un mismo servidor puede alojar varias webs con su correo y bases de datos dentro de la misma máquina, a cada uno de esos espacios los llamamos hosting

¿Que es un anuncio y como se hacen?

Los anuncios son imágenes con presentaciones que se diseñan con el objetivo de promover y vender un servicio o producto, estos se realizan con el fin de que genere interés e impacten en nuestro posible cliente para así redireccionarlos a un lugar deseado como puede ser una LandinPage, hay varias plataformas para subir nuestros anuncios y cada una de ellas puede ser más conveniente según un producto o servicio, como ejemplo en el caso de promover un restaurante sería poco inteligente que el anuncio impacte en otra provincia, ya que queremos que el cliente se encuentre en una cercanía geográfica, cosa que con un producto no habría problema. Esta gestión se ejecuta con anterioridad para lograr la estrategia más adecuada para vos. El costo de lo que cobra la plataforma corre por parte del cliente.
